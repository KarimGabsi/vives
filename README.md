# VIVES COURSES
https://docusaurus.io/docs/installation

## Install
1) Node.js
2) If brand new:
   npx create-docusaurus@latest my-website classic
   (npx create-docusaurus@latest courses classic)
   If pulled from GitHub:
   cd courses
   npm install

## Build
1) npm run build


## Run
1) cd courses
2) npm run start
