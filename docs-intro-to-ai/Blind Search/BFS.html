Breadth First Search

Pseudocode​

1) QUEUE (path containing the root)

2) WHILE (QUEUE is not empty AND goal is not reached)

  • Remove the first path from the QUEUE
  • Create new paths (to all children)
  • Reject the new paths with loops
  • Add the new path to the back of QUEUE

3) IF (goal reached)

  • THEN SUCCESS
  • ELSE FAILURE

Criteria​

b = depth of the tree
d = (average) branching factor of the tree
m = depth of the shallowest solution

Speed & Memory​

Time and State Complexity:O(bm)Time\ and\ State\ Complexity: O(b^m)Time and State Complexity:O(bm)
